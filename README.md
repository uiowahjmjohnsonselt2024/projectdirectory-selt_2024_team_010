# Project Setup

## Requirements

Make sure you have the following versions installed:

- **Ruby**: 2.6.10
- **Bundler**: 1.17.3

If you need to install a specific version of Bundler, you can run:

```bash
gem install bundler -v 1.17.3
```

## Setup
```bash
bundle install
```
```bash
rake db:setup
```

You will also need to install Redis for job handling:
- **Redis**: 7.4.6
